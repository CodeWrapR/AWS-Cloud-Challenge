AWS Day 1: Learn and Test

Flashcard Learning

  • What is Cloud Computing?
    Cloud computing is delivering computing services over the internet including servers, storage, databases, networking, software, analytics, and more.
  • What is AWS?
    AWS is Amazon‚Äôs cloud platform offering over 200 fully featured services from data centers globally.
  • Benefits of Cloud:
    Pay-as-you-go, scalability, flexibility, speed, cost-efficiency, and global reach.
  • Cloud Economics:
    Optimizing resources, reducing CapEx, and aligning spend with usage.
  • Cloud Design Principles:
    Design for failure, implement elasticity, automate, decouple components, use managed services.
  • IaaS, PaaS, SaaS:
    • IaaS: You manage the OS, middleware, runtime, and data (e.g., EC2).
    • PaaS: You manage just the app and data (e.g., Elastic Beanstalk).
    • SaaS: Provider manages everything (e.g., Gmail, Salesforce).
