AWS Day 1: Learn & Quiz

Flashcard Learning

  • Cloud Computing: Internet-based computing services (storage, compute, databases, etc.).
  • AWS Overview: Cloud platform by Amazon with 200+ services globally available.
  • Benefits of Cloud: Pay-as-you-go, scalability, flexibility, global presence.
  • Cloud Economics: Cost optimization, reduce CapEx, align costs with usage.
  • Design Principles: Design for failure, use automation, leverage managed services, build loosely coupled systems.
  • Delivery Models:
    • IaaS: EC2, manage OS and runtime.
    • PaaS: Elastic Beanstalk, manage code/app only.
    • SaaS: Fully managed (e.g., Gmail, Salesforce).
